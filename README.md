

# 📊 Data Analyst Internship Portfolio – Sana Ansari

Welcome to my Data Analyst Internship Portfolio. This repository showcases my end-to-end journey through data analysis — from raw data processing to advanced insights, dashboarding, and business storytelling.

Throughout this internship, I worked on real-world datasets and developed strong skills in data cleaning, SQL, visualization, and analytical thinking.

---

# 🧭 Internship Overview

This portfolio is structured into four key phases:

1. Data Immersion & Wrangling
2. Exploratory Data Analysis (EDA) & Business Intelligence
3. Deep-Dive Analysis & Interactive Dashboarding
4. Data Storytelling & Statistical Validation

Each phase builds upon the previous one to simulate a real-world data analytics workflow.

---

# 🧩 TASK 1: Data Immersion & Wrangling

## 🎯 Objective

To understand the dataset, assess data quality, and prepare clean, structured data for analysis.

## 🔧 Steps Performed

1. **Data Access & Familiarization**

   * Explored dataset structure, features, and data types
   * Understood business context and data sources

2. **Data Quality Assessment**

   * Identified missing values, duplicates, and inconsistencies
   * Checked data validity and formatting issues

3. **Data Cleaning & Transformation**

   * Handled missing/null values
   * Removed duplicates
   * Standardized formats and data types
   * Created new derived columns where necessary

## 📦 Deliverables

* Cleaned dataset
* Data preprocessing documentation

---

# 📊 TASK 2: Exploratory Data Analysis (EDA) & Business Intelligence

## 🎯 Objective

To uncover patterns, trends, and insights using statistical analysis and SQL.

## 🔧 Steps Performed

1. **Descriptive Statistics & Univariate Analysis**

   * Calculated mean, median, mode, and distribution metrics
   * Created histograms and bar charts

2. **SQL for Business Questions**

   * Solved real-world business queries such as:

     * Top-performing products by revenue
     * Monthly user growth trends
   * Used filtering, aggregation, and joins

3. **Multivariate Analysis & Correlation**

   * Analyzed relationships between variables
   * Built scatter plots and correlation visuals

4. **Static Dashboard Mock-up**

   * Designed a dashboard layout showcasing key metrics and KPIs

## 📦 Deliverables

* EDA report
* SQL queries with results
* Dashboard mock-up

---

# 📈 TASK 3: Deep-Dive Analysis & Interactive Dashboarding

## 🎯 Objective

To solve complex business problems and build an interactive dashboard.

## 🔧 Steps Performed

1. **Define Core KPIs**

   * Conversion Rate
   * Customer Churn Rate
   * Average Order Value
   * Provided formulas and business relevance

2. **Deep-Dive Analysis**

   * Performed advanced analysis such as:

     * Cohort Analysis (retention trends)
     * Funnel Analysis (user drop-off)
     * Segmentation Analysis (customer grouping)

3. **Interactive Dashboard Build**

   * Built dynamic dashboards using BI tools (Power BI/Tableau)
   * Enabled filtering and drill-down capabilities

---

# 📖 TASK 4: Data Storytelling & Statistical Validation

## 🎯 Objective

To communicate insights effectively and validate findings using statistics.

## 🔧 Steps Performed

1. **Business Storytelling**

   * Structured analysis into a clear narrative:

     * Problem → Analysis → Insight → Recommendation

2. **Final Presentation Development**

   * Designed a professional slide deck
   * Focused on clarity, visuals, and business impact

3. **Statistical Testing**

   * Formulated hypotheses
   * Performed tests such as:

     * T-test
     * Chi-square test
   * Interpreted p-values and confidence intervals

## 📦 Deliverables

* Final presentation deck
* Hypothesis testing summary
